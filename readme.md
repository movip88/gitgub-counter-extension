Para realizar una consulta sobre un repositorio se llama a barra de esta forma
    -> http://localhost:3000/{owner_repo}/{repo_name}

Ejemplo
    -> http://localhost:3000/movip88/Ajax-NativeScript
    
Resultado
    {
        "md": 1,
        "DS_Store": 1,
        "css": 8,
        "js": 18,
        "ts": 18,
        "html": 9,
        "json": 1
    }
